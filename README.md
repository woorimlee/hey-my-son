## hey-my-son
자녀가 게임을 플레이했는지 확인해보자.

(개발중)


### process flow
1. 데이터 수집 (롤, 배그, 오버워치, 메이플스토리)
2. 전적 변화 확인 (csv 형식으로 기록 누적할 예정)
3. 게임을 얼마나 더 플레이했는지(전적차이 + 대략 플레이한 시간) 확인해서
4. 변화 있을 때마다 카톡으로 보낼예정.
